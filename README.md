# grpc-practice
- 本專案為 grpc 相關學習（包括 protocol buffer 的使用、Makefile 等等），並且分析比較其與以往 restful 不同處
- 目前主要以 C++ 作為主要語言，未來會加上 nodejs 版本、go 版本做實際使用（實際與現今 restful 架構做比較）
